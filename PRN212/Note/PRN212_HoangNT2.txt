0. SDK LÀ GÌ? .NET SDK LÀ GÌ?
- SDK - SOFRWARE DEVELOPMENT KIT - LÀ TỪ DÙNG CHUNG CHO MỌI NGÔN NGỮ LẬP TRÌNH
- NÓ LÀ BỘ CÔNG CỤ BỘ ĐỒ CHƠI, CÁC THƯ VIỆN, CÁC TOOL, CÁC APP (ĐƯỢC HÃNG LÀM RA NGÔN NGỮ LẬP TRÌNH) DÙNG CHO DÂN DEV GIÚP DÂN DEV LÀM APP
- LÀM APP THÌ CẦN 1 NGÔN NGỮ LẬP TRÌNH ĐỂ VIẾT CÁC LỆNH ĐỂ RA LỆNH CHO MÁY LÀM ĐIỀU GÌ ĐÓ. NNLT CẦN THÊM CÁC ĐỒ CHƠI... ĐỂ RA LỆNH CHO MÁY LÀM ĐIỀU GÌ ĐÓ. DÙNG NNLT + ĐỒ CHƠI (SDK) SẼ GIÚP TẠO RA APP VÀ RUN APP
- MUỐN HỌC 1 NNLT NÀO ĐÓ ĐỂ VIẾT APP GÌ ĐÓ TRƯỚC TIÊN TA CẦN CÀI SDK.

* VỚI JAVA, TA CẦN JAVA SDK - GỌI TẮT LÀ JDK (JAVA DEVELOPMENT KIT)

* SDK VS RUNTIME
- SDK > RUNTIME
- CÀI SDK TỨC LÀ CÀI LUÔN RUNTIME. RUNTIME CÓ THỂ CÀI RIÊNG MÀ KHÔNG CẦN SDK (PHẦN DÙNG CHO DÂN DEV)
- RUN TIME: BỘ THƯ VIỆN, CÂU LỆNH, MÔI TRƯỜNG GIÚP APP MÌNH VIẾT RA TƯƠNG TÁC ĐC VỚI OS; NHỎ GỌN, KO CẦN PHẦN TOOL DÀNH CHO DÂN DEV
- SDK: KÈM THÊM RUNTIME, KÈM THÊM CÁC CÔNG CỤ CHO DÂN DEV: BIÊN DỊCH, DEBUG, GENERATE CODE DOCUMENT, KIỂM THỬ APP/CODE, ĐÓNG GÓI
(.JAR, .WAR) .DLL .EXE
